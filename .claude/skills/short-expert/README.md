## 🚀 创作流程概览

### 第一步：标题与核心创意
**目标**：确定项目基本信息，验证商业可行性  
**输出**：`00_项目信息.md`  
**关键检查**：标题网感、卖点清晰、爽点明确

### 第二步：故事简介与梗概
**目标**：完成投流简介 + 四幕梗概  
**输出**：`01_故事梗概.md`  
**关键检查**：四幕结构完整、情绪走向清晰

### 第三步：人物小传
**目标**：为每个重要角色创建独立角色卡  
**输出**：`02_角色卡/*.md`  
**关键检查**：人设极致化、人物弧光清晰

### 第四步：完整故事线
**目标**：生成全剧完整故事线，确立剧情逻辑与连贯性  
**输出**：`03_完整故事线/第X阶段故事线.md`（按阶段分文件）  
**关键内容**：每集300-500字的完整故事情节  
**侧重点**：逻辑性、连贯性、饱满度

### 第五步：分集大纲
**目标**：逐集拆解，包含爽点分析和付费点验证  
**输出**：`04_分集大纲/第XX集_大纲.md`  
**关键内容**：场景拆解、情绪曲线、卡点设计、付费点验证

### 第六步：剧本正文
**目标**：逐集输出可拍摄剧本  
**输出**：`05_剧本正文/第XX集_剧本.md`  
**特别提醒**：每完成10集，必须进行一致性检查

## ⚡ 核心要点

### 上瘾三要素
1. **永远不让心理闭环完成** - 解决一个问题，立刻抛出新问题
2. **高频情绪奖励** - 每30秒一个情绪波动点
3. **递进式悬念设计** - 悬念强度不断升级，信息分批释放

### 三层爽点体系
1. **动物级爽点**：生存、暴力、两性（底层驱动）
2. **人性级爽点**：优越感、秩序回归、公平正义（心理归属）
3. **情绪拉扯**：爽→虐→更爽→更虐→超级爽（高黏性秘诀）

### 商业化铁律
- 黄金3秒必须有钩子
- 每90秒一个小高潮
- 每集结尾必有卡点（毒性等级★★★以上）
- 付费点前必须有足够的情绪积累
- 悬念接力：解决30%旧悬念 → 制造100%新悬念
- 人设极致化：坏要坏到丧心病狂，惨要惨到惨绝人寰

## ✅ 一致性保障机制

**触发时机**：每完成10集剧本

**检查维度**：
1. 剧情逻辑一致性
2. 人物状态一致性
3. 内容重复性检查
4. 伏笔与回收一致性
5. 情绪与悬念连贯性

**输出**：`07_一致性检查报告/第XX-XX集_检查报告.md`

详见 [CONSISTENCY.md](CONSISTENCY.md) 一致性检查方法论

## 💡 使用建议

1. **严格遵循流程**：每步完成并获得确认后，才能进入下一步
2. **重视完整故事线**：第四步是保证全剧连贯性的核心
3. **坚持一致性检查**：每10集必须检查，不能积累问题
4. **人设极致化**：惨要极致、强要极致、宠要极致
5. **卡点设计毒性**：每集结尾卡点毒性必须达到★★★以上
